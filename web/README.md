## Halo Command Lattice

Strategic Halo operations dashboard built with Next.js App Router and Tailwind CSS.

### Local Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the live interface with hot reload enabled.

### Production Build

```bash
npm run build
npm run start
```

### Tech Stack
- Next.js 16 (App Router, TypeScript)
- Tailwind CSS via `@tailwindcss/postcss`
- Geist Sans & Mono (Next.js font optimization)

### Key Files
- `src/app/page.tsx` — command lattice layout and interactive modules
- `src/app/layout.tsx` — root layout, fonts, metadata
- `src/app/globals.css` — halo-inspired global styling
